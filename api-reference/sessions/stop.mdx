---
title: "Stop a session"

---

<Note>
  `POST https://api.uip.digital/v1/sessions/{id}/stop` · Auth: `Bearer` API key
</Note>

Ends a session paused at a [gate](/core-concepts/primitives#gates-pause-for-a-decision)
(`status: awaiting_decision`) without running its remaining steps — the decision
counterpart to [continue](/api-reference/sessions/continue). The session moves to
`stopped` and a `session.stopped` webhook fires.

## Path parameters

<ParamField path="id" type="string" required>The session id.</ParamField>

## Behavior

- Valid only when the session is `awaiting_decision`.
- Owner-only: the API key's business must own the session.
- Already-completed steps remain verified and billed; remaining steps are skipped.

<ResponseExample>
```json 200 OK
{
  "id": "sess_abc123",
  "status": "stopped",
  "current_step": 0,
  "step_count": 2
}
```
</ResponseExample>

`404` unknown/foreign session · `409`/`400` if the session isn't at a gate. See
[Errors](/api-reference/errors).
